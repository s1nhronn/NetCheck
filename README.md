# NetCheck

Простой Python-скрипт для проверки интернет-соединения через `ping`.

Проверяет:
- доступность роутера (шлюза)
- доступность внешнего хоста

Результат выводится в консоль и записывается в `internet_check.log`.

## Запуск

```bash
python main.py
```

## Настройка

Измените параметры в начале файла:
```python
GATEWAY = "192.168.1.1"  # IP роутера
TARGET = "8.8.8.8"       # внешний хост
INTERVAL = 5             # интервал (сек)
LOGFILE = "internet_check.log"
```

## Логика
* роутер недоступен = проблема в локальной сети / кабеле
* роутер доступен, но нет интернета = проблема у провайдера
* всё доступно = интернет работает

## Требования
* Python 3
* доступная команда ping
